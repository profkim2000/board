  1. e 키를 눌러 extrude를 시작하고 x 키를 누른다.

  3. e 키를 눌러 extrude를 시작하고 ctrl + x 키를 누른다.
  4. x 키를 눌러 extrude를 시작하고 ctrl + x 키를 누른다.
