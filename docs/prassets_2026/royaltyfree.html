Eugene's Board

저작권 걱정없이 다운로드받아 사용할 수 있는 사이트들

주의사항: 모든 자료에 대해 반드시 건별로 라이선스 확인해야 함.

업데이트: 2026. 2. 28.


관련 용어

public domain

  • 공적 사용 라이선스. 누구나 무료로 사용 가능함. 상업적 사용 가능 포함.
  • CC0: public domain 표기법 중 하나.
  • 나무위키 설명: 링크
  • 라이선스 관련 사이트: Creative Commons
  • 아래 사이트에서 CC0 라이선스로 공시한 걸 보고 다운로드 받아 사용했는데, 업로드한 사람이 다른 사람의 작품을 속여서 업로드한 거라면 소송에 휘말릴 수 있음
    • 업로드한 이미지가 한 두개만 있는 사람이 올린 자료는 가급적 피한다.
    • 한 사람이 여러 개 올렸어도 화풍이 확연히 다른 이미지는 가급적 피한다.

attribution

  • 원래는 속성, 원인, 책임 등의 뜻을 가지는데, 라이선스 분야에서는 "출처 표시" 라는 뜻으로 사용된다.
  • 예: without attribution = "출처 표시없이"

commercial use

  • 상업적 사용
  • 사용 예(iconmonstr의 Q&A 발췌):

pixabay

  • URL: https://pixabay.com
  • 사진, 그림, 비디오, 오디오, 음향효과 등 제공.
  • 자료 제공자가 처음부터 라이선스를 요청하지 않을 것(royalty free)에 동의하고 올리기 때문에 문제없이 사용할 수 있음
  • 사진 업로드 승인이 잘 나지 않음 = 고퀄리티 사진 제공
  • 검색어는 영어만 가능

공유마당

  • URL: https://gongu.copyright.or.kr
  • 사진, 비디오, 영상, 글꼴 등 제공
  • 저작권이 만료된 오래된 자료나 자료 제공자가 공공 사용을 전제로 하고 업로드 하는 경우가 대부분
  • 주의 사항: 자료 제공자가 다양한 라이선스 요청하고 있음.(예: 음악을 동영상 제작에 사용 시, 작곡자를 동영상에 몇 초 이상 표시할 것) 따라서 사용 전에 라이선스를 반드시 확인해야 함.

퍼블릭 벡터

  • URL: https://publicdomainvectors.org
  • CC0의 벡터 이미지만 공유하는 사이트
  • 위 아래 광고

아이콘몬스터

  • URL: https://iconmonstr.com
  • 아이콘 다운로드 받아 사용
  • 변형없이 그대로 사용해야 한다는 라이선스
  • 단순 다운로드가 아니라 크기, 선색 등 변경 가능

눈누

  • URL: https://noonnu.cc
  • 글꼴을 모아 보여주고 체험할 수 있는 사이트
  • 이 사이트의 모든 글꼴이 다 무료는 아님
  • 무료였던 글꼴이 유료로 바뀌는 경우도 있으니 사용할 때마다 확인하고 라이선스 파일을 다운로드 받아 잘 저장해야 함
  • 글꼴은 네이버 글꼴을 사용하는 것이 저작권상 안전하다
  • 한컴 오피스에서 제공하는 글꼴을 한컴 오피스 외 (예: 파워포인트)에서 사용은 가능하지만, 라이선스 위반임.

유튜브 스튜디오

  • 주의사항: 음악, 효과음등 제공하지만 이 애셋들을 이용해 만든 영상은 유튜브에만 올려야 함. 유튜브 스튜디오에서 제공하는 애셋을 이용해 만든 영상을 유튜브 외에 올리면 라이선스 위반.
