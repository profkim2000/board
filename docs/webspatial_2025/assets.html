QGIS 마우스 기본 사용법

  • 화면 이동 : 마우스 휠 누르고 움직이기

QGIS 점(point) 표시한 shp 파일 만들기

다음 순서대로 진행한다.

1. 새 프로젝트 만들기

[새 프로젝트] 아이콘을 클릭해 새 프로젝트를 만든다.


2. OpenStreeMap 밑에 깔기

XYZ Tiles > OpenStreetMap 을 더블클릭한다.


3. shp 파일을 생성하고, 그걸 이용하는 레이어 추가

[새 세이프파일 레이어] 툴바를 클릭한다.

[…] 버튼을 클릭한다.

shp 파일명을 정해준다.

위 그림 부분을 다음과 같이 세팅한다.

  • 파일 인코딩: UTF-8
  • 도형 유형: 포인트
  • 추가 차원: 없음
  • 좌표계: EPSG:4326 - WGS 84

다음의 방법으로 필드를 추가한다.

  • 이름: name

이렇게 추가한 필드가 각 점의 정보를 저장하는데 사용된다.

점을 하나 추가할 때마다, 필드 목록에 있는 id와 name을 입력해야 한다.


4. 레이어 편집 모드로 변경


5. 점 추가

점 추가 버튼을 누른 다음,

원하는 위치에 점을 추가할 수 있다.

  • [1] 클릭한 위치에 점이 추가되고
  • [2] 점이 추가되자마자 위와 같은 속성 창이 나타나서 자료를 입력할 수 있다. id값은 점을 찍을 때마다 중복된 값이 입력되지 않도록 1, 2, 3, 4, 5, … 으로 알아서 점마다 다른 값을입력해야 한다.
    name 항목은 조금 전에 추가한 그 항목이다.
