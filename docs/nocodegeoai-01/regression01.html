실습 4 - 선형회귀 다항회귀

선형회귀

선형회귀의 앞 부분은 보스턴 주택가격 실습을 이용합니다. 보스턴 주택가격을 먼저 완성해 주세요.


선형회귀 모델이 만든 그래프의 기울기와 절편 보기

  1. Data에서 “Data Table” 위젯을 추가하고, “Linear Regression” 노드와 연결한 다음, “Data Table” 위젯을 더블클릭한다.

  2. Linear Regression 모델에서 학습한 결과가 나타난다.


1번이 Intercept인데 절편을 나타낸다.

절편은 x의 값이 0이 되는 곳의 y값 = y축과 직선이 만나는 곳 이다.


그 아래 범죄율, 주거지 비율, 상업지 비율, … 등은 집값에 영향을 미치는 요소들의 가중치이다.

그래서 이걸 식으로 나타내면 다음과 같다.

집값 = -0.108011 * 범죄율 + 0.0464205 * 주거지 비율 + 0.0205586 * 상업지 비율 + ….. + -0.524758 * 하위계층 비율 + 36.4595

값이 크면 클수록 집값에 영향을 많이 미친다.

범죄율의 경우 음수인데, 범죄율이 높을수록 집값이 떨어진다고 학습한 결과이다.


그래프로 직선 보기

  1. Visualize에서 “Scatter Plot” 위젯을 추가한다.

  2. “Data Table” 노드와 방금 전 추가한 “Scatter Plot” 노드를 연결한다.

  3. “Predictions” 노드와 방금 전 추가한 “Scatter Plot” 노드를 연결한다.

  4. Scatter Plot 창이 나타난다.

  5. “Show regression line” 옵션을 체크표시가 나타나도록 변경한다.

이제 Linear Regression 모델이 학습하면서 만든 그래프의 직선이 나타난다.

Axis X의 값을 범죄율, 주거지 비율 등으로 변경해 가며 확인한다.


다항회귀

데이터 만들기

  1. Data 에서 “Paint Data” 위젯을 추가하고 더블클릭한다.

  2. Paint Data 창이 나타난다.

  3. 오른쪽 그래프 부분에 마우스로 위 그림과 비슷하게 그린다.

이제 우리의 데이터는 직선 그래프로는 예측하기 어려운, 곡선형으로 예측해야 하는 데이터로 만들어졌다.


데이터를 시각화해서 보기

  1. Visualize에서 “Scatter Plot“을 추가하고 “Paint Data” 노드와 연결한다.

  2. scatter plot 형식으로 방금 전 “그려 넣은” 데이터의 모양을 볼 수 있다.


데이터의 Feature와 Target 구분하기

Paint Data는 데이터를 쉽게 만들어 주지만 target을 지정해 주지 못한다. 그래서 Select Columns 위젯을 이용해 target을 지정해 준다.

  • Feature: 독립변수, X, target 값을 결정하는 요인.

  • Target: 종속변수, Y, 주택가격 등 feature 들에 의해 결정된 값.


  1. Transform에서 “Select Columns” 위젯을 추가한다.

  2. Select Columns 창이 나타나면 Features 항목에서 “Y” 를 Target 항목으로 끌어내린다.(드래그 앤 드롭)

  3. 닫기 버튼을 누른다.

이제 X축의 값이 feature이고, Y축의 값이 target이 되었다.


애드온 추가 - Educational 애드온

다항 회귀를 위한 Polynomial Regression 위젯을 설치하기 위해 “Educational” 애드온을 설치한다.

  1. 메뉴에서 “Options” 를 선택한다.

  2. “Add-ons…“를 선택한다.


  1. 애드온 Installer 창에서 “Educational“을 선택한다. (체크가 된 상태여야 함)

  2. OK 버튼을 누른다.


애드온이 설치된다. 약 1분 정도 소요된다.


설치가 끝나면 다시 시작하겠냐고 물어본다. “OK“를 눌러 다시 시작한다.


Polynomial Regression 위젯 추가

  1. “Scatter Plot” 노드는 삭제한다.

  2. 아마도 맨 밑에 있을 Educational에서 “Polynomial Regression” 위젯을 추가하고 “Paint Data” 노드와 연결한다. 그리고 더블클릭한다.

  3. Polynomial degree의 값을 “2“로 세팅한다. 데이터와 매우 비슷한 곡선 모양으로 학습된 것을 확인할 수 있다.


다항 위젯 성능 테스트

  1. Evaluate 에서 “Test and Score” 위젯을 추가한다.

  2. “Polynomial Regression” 노드와 “Test and Score” 노드를 연결한다. 이 때 Learner 로 연결되었는지 = 연결선에 Learner라고 표시되는지 확인한다. 만약 이렇게 나오지 않으면 선을 더블클릭해 Learner라고 나오도록 수정한다.

  3. “Select Columns” 노드와 “Test and Score” 노드를 연결한다. 이 때 Data 로 연결되었는지 = 연결선에 Data 라고 표시되는지 확인한다.

  4. R2의 값을 확인한다. 0.974면 매우 훌륭한 모델이다.


Linear Regression과의 성능 비교

  1. Model에서 “Linear Regression” 위젯을 추가하고 “Select Columns“와 연결하고, “Test and Score“와도 연결한다.

  2. “Test and Score” 노드를 더블클릭한다.


두 모델, “Polynomial Regression“과 “Linear Regression“의 성능 차이가 한 눈에 나타난다.

이런 데이터의 경우 Linear Regression의 R2 값은 음수로 매우 성능이 좋지 않음을 알 수 있다.
