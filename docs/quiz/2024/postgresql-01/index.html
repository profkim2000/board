    복습문제

    1. 다음 중 데이터베이스의 특징이 아닌 것은?

    1. 자료를 체계적으로 저장한다.
    2. 자료를 저장하기 쉽고 읽기 쉽게 관리한다.
    3. 자료를 모아 놓은 것은 모두 데이터베이스이다.
    4. 공사 ERP가 데이터베이스의 한 예이다.

    2. DBMS의 약자로 맞는 것은?

    1. DataBase Management System
    2. DB Manager System
    3. DB Management System
    4. DataBase Manager System

    3. postgreSQL에 대한 설명 중 틀린 것은?

    1. open source로 개발된다.
    2. 개인은 무료, 공공기관은 비용을 지불해야 한다.
    3. 공간정보에 특화된 기능을 제공한다.
    4. 포스트그레스큐엘이라고 읽는다.

    4. 다음 중 DBMS가 아닌 것은?

    1. postgreSQL
    2. MS SQLServer
    3. MySQL
    4. DBeaver

    5. 다음 중 공간정보 데이터베이스에서 지원하는 기능이 아닌 것은?

    1. 다양한 좌표계 지원
    2. 3차원 스캐닝 기능 지원
    3. 공간정보 자료(점, 선, 면)을 자유롭게 저장
    4. 공간정보 계산 기능 지원

    6. 다음 중 EPSG에 대한 설명으로 틀린 것은?

    1. 석유 산업계에서 탐사를 위해 표준화된 좌표 체계가 필요해 정리한 코드들이다.
    2. EPSG의 원 뜻은 유럽 석유 조사 그룹이다.
    3. 각 좌표계 코드 표현식은 EPSG-4326 과 같은 식이다.
    4. 한국의 좌표 체계도 표함되어 있다.
