6. pgAdmin에 대한 설명으로 틀린 것은?

  1. postgreSQL에 접속해 postgreSQL을 관리하거나 SQL문을 실행하는 데 사용하는 텍스트 기반의 클라이언트 프로그램

  3. 유료 버전을 구입하면 그래픽 UI도 사용할 수 있다.
  4. 관리자 전용 프로그램이다.

);
