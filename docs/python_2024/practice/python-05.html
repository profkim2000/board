2. 구구단을 gugudan.txt 파일로 출력하는 코드

답(안) 
file = open("gugudan.txt", 'w')

i = 2
while i <= 9 :
    j = 1    
    while j <= 9 :
        a = "{0} * {1} = {2}\n".format(i, j, i*j)        
        file.write(a)

        j += 1
    i += 1

file.close()
