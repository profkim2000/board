3. OpenLayers 작업

1. d:\openlayers\proj4 폴더를 만든다.


2. 이 폴더에서 PowerShell을 연다.

  1. 빈 공간을 Shift 키를 누른 채로 마우스 오른쪽 클릭한다.

  2. “여기에 PowerShell 창 열기” 메뉴를 선택한다.


3. openLayers 프로젝트 생성

powerShell 내에서 다음을 실행해 openLayers 프로젝트를 생성한다.

프로젝트명은 “map-trip” 이다.

npm  create  ol-app  map-trip

“map-trip” 폴더가 생성된다.


4. map-trip 폴더에서 visual studio code 실행

  1. 탐색기에서 “map-trip” 폴더로 이동한다.

  2. “map-trip“에서 빈 공간을 오른쪽 클릭한 후 “Code(으)로 열기” 메뉴를 선택한다.

“작성자를 신뢰합니까” 라는 질문에 “예. 작성자를 신뢰합니다.“를 선택한다.


5. OpenLayers 실행

  1. visual studio에서 “보기” 메뉴를 선택한다.

  2. “터미널“을 선택한다.

  3. 터미널 창이 아래쪽에 열린다.


  1. 터미널 창에서 npm start 를 입력하고 엔터친다.

위와 같이 실행되는 내용이 나타난다.

이 중 웹 주소가 보이는데, 이 주소를 아래 그림과 같이 웹브라우저 url 창에 입력한다.


좀 전에 npm start라고 한 것으로 인해 내 컴퓨터에서 내부적으로 openLayers 서버가 실행되었고, 그 내부 서버에 접속했더니 위와 같이 지도가 나타났다.
이 지도는 Open Street Map에서 무료로 제공하는 지도로 openLayers에서 기본으로 사용하는 지도이다.
웹지도의 기본 기능인 줌, 이동 등이 제공된다.


6. GeoServer가 실행 중인지 확인

GeoServer가 실행 중인지, 레이어의 이름이 “proj4:trip“인지 확인한다.


7. 우리가 하려는 것: 지도 위에 여행가고 싶은 곳 표시하기

main.js 파일 수정

  1. main.js 파일을 선택한다.

  2. main.js 파일을 아래의 코드로 수정한다.

import 'ol/ol.css';
import Map from 'ol/Map';
import View from 'ol/View';
import TileLayer from 'ol/layer/Tile';
import OSM from 'ol/source/OSM';
import TileWMS from 'ol/source/TileWMS';
import { fromLonLat } from 'ol/proj';

// 1. OpenStreetMap 배경지도 레이어 생성
const osmLayer = new TileLayer({
  source: new OSM(),
});

// 2. GeoServer WMS 레이어 생성
const geoserverWmsLayer = new TileLayer({
  source: new TileWMS({
    // GeoServer의 WMS 서비스 엔드포인트 주소
    url: 'http://localhost:8080/geoserver/proj4/wms',
    params: {
      'LAYERS': 'proj4:trip',       // 작업공간:레이어이름
      'TILED': true,                // 타일 방식으로 성능 최적화
    },

    // OSM 지도(EPSG:3857)와 좌표계를 맞춥니다.
    projection: 'EPSG:3857', 
  }),
});

// 3. 지도 객체 생성 및 레이어 결합
const map = new Map({
  target: 'map', // HTML에서 지도가 들어갈 div의 id
  layers: [
    osmLayer,          // 밑에 깔릴 OSM
    geoserverWmsLayer, // OSM 위에 얹어질 GeoServer WMS
  ],
  view: new View({
    // 지도의 초기 중심점 (경도, 위도 순서로 입력 후 OSM 좌표계로 변환)
    // 예: [127.0, 37.5]는 대한민국 서울 근처입니다. 데이터 위치에 맞게 수정하세요!
    center: fromLonLat([127.0, 37.5]), 
    zoom: 7, // 초기 줌 레벨
  }),
});
  1. main.js 파일을 저장한다.

아직까지는 웹브라우저에 별다른 변화도 없고, 오히려 화면이 아무 것도 표시되지 않는 흰 화면이 되어 있을 거다.


index.html 파일 수정

  1. index.html 파일을 선택한다.

  2. 표시한 부분을 다음의 코드로 수정한다.

<div id="map" style="width: 100%; height: 100vh;"></div>

최종적으로 index.html 파일의 전체 내용은 다음과 같다.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <link rel="icon" type="image/x-icon" href="https://openlayers.org/favicon.ico" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Using OpenLayers with Vite</title>
  </head>
  <body>
    <div id="map" style="width: 100%; height: 100vh;"></div>
    <script type="module" src="./main.js"></script>
  </body>
</html>

웹브라우저 타이틀이 마음에 들지 않는다.

내가 원하는 것으로 바꿔보자.

  2. title 부분을 수정한다. 주의할 점은 <title>과 </title> 부분은 지우면 안되고 그 둘 사이의 문자만 수정해야 한다는 것이다.
