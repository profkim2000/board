토공량 분석

"토공량 분석"을 선택한다.


"영역그리기" 버튼을 누른다.


  1. 토공을 원하는 영역을 점을 찍어 그린다.

  2. "그리기 완료" 버튼을 누른다.


계획 높이의 값이 영역을 잡은 부분의 평균고도의 값으로 바뀌었다는 메시지. 확인 버튼을 누른다.


  1. 평균높이를 확인한다.

  2. 계획높이가 위 메시지처럼 평균높이로 바뀐 것을 확인한다.


  1. 계획높이를 평균높이보다는 낮은 높이로 바꾼다.(절토)

  2. "영역그리기" 버튼을 누른다.

  3. 토공 시뮬레이션 결과를 확인한다.

  4. 토공량 분석 결과를 확인한다.

  5. 확인이 끝나면 "초기화" 버튼을 클릭해 시뮬레이션을 종료한다.


사용자 모델

"사용자모델"을 클릭한다.


  1. "샘플다운로드" 를 누른다.

  2. "아파트 샘플을 다운로드 받는다.


  1. "불러오기" 버튼을 누른다.

  2. "glb"를 누른다.

  3. 다운로드받은 아파트 샘플 파일을 선택해 불러온다.

  4. "확인" 버튼을 누른다.


  1. 아파트 모델을 위치할 지점을 클릭한다.

  2. "지점선택완료" 버튼을 누른다.


결과
